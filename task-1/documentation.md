# Task 1

I’m still new to C++, so this task helped me understand a few basic things:

- Data sizes are really important. If the CPU can handle 2 bytes max, that means the biggest type here is 16 bits because 1 byte is 8 bits.

- Namespaces are useful when two functions have the same name (`sum`) but are meant for different types.

- `unsigned int` does not go below 0. If you do `0 - 1`, it wraps around to a very large number.

I kept the code simple on purpose so it’s easy to read and explain.
